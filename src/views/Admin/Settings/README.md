# 系统设置页面

## 功能说明

系统设置页面允许管理员配置平台的各种参数，包括：

### 1. 网站信息设置
- 网站名称
- 网站Logo
- 网站Favicon

### 2. SEO设置
- SEO标题
- SEO关键词
- SEO描述

### 3. 上传限制设置
- 最大文件大小（MB）
- 允许的文件格式

### 4. 下载限制设置
- 普通用户每日下载次数限制

### 5. 水印配置设置
- 水印文字
- 水印透明度
- 水印位置
- 实时预览效果

### 6. 支付方式设置
- 微信支付开关
- 支付宝支付开关
- 积分充值功能开关
- VIP自动续费开关

## 功能特性

### 表单验证
- 所有表单字段都有实时验证
- 必填字段验证
- 长度限制验证
- 数值范围验证

### 保存功能
- 支持保存所有设置
- 保存前验证所有表单
- 保存成功后显示提示
- 保存后立即生效

### 重置功能
- 支持重置所有设置为默认值
- 重置前二次确认
- 重置后自动重新加载设置

### 用户体验
- 使用标签页组织不同设置模块
- 表单字段有详细的提示说明
- 水印配置有实时预览效果
- 支持暗黑模式

## API接口

### 获取系统设置
```
GET /api/v1/admin/settings
```

### 更新系统设置
```
PUT /api/v1/admin/settings
```

### 重置系统设置
```
POST /api/v1/admin/settings/reset
```

## 路由配置

```
/admin/settings
```

## 使用说明

1. 登录管理后台
2. 点击侧边栏的"系统设置"菜单
3. 在不同的标签页中修改相应的设置
4. 点击"保存设置"按钮保存修改
5. 如需恢复默认设置，点击"重置为默认"按钮

## 注意事项

1. 所有设置修改后需要点击"保存设置"按钮才会生效
2. 重置操作会将所有设置恢复为默认值，请谨慎操作
3. 水印配置有实时预览功能，可以直观看到效果
4. 文件大小和下载次数限制仅对普通用户生效，VIP用户不受限制
